Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using elements represented by tags.

HTML elements are the building blocks of HTML pages. Elements are represented by tags such as <h1>, <p>, and <div>.

HTML Headings

HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading, and <h6> defines the least important heading.

HTML Paragraphs

HTML paragraphs are defined with the <p> tag. A paragraph always starts on a new line and browsers automatically add some space (margin) before and after a paragraph.

HTML Lists

HTML supports ordered (numbered) and unordered (bulleted) lists. Ordered lists are created with the <ol> tag, and unordered lists with the <ul> tag. List items are defined with the <li> tag.

HTML Links

HTML links are defined with the <a> tag. The href attribute specifies the URL of the page the link goes to.

Visit Example

HTML Images

HTML images are defined with the <img> tag. The src attribute specifies the path to the image, and the alt attribute provides alternative text for the image.

HTML Tables

HTML tables are defined with the <table> tag. A table is divided into rows (<tr>), and each row is divided into cells (<td>). Table headers are defined with the <th> tag.

Name Age City
John 45 New York
Jane 50 Los Angeles

HTML Forms

HTML forms are used to collect user input. The <form> element defines a form, and various form elements like <input>, <textarea>, and <button> are used to create form fields.
