Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. It supports multiple programming paradigms and is widely used for web development, data analysis, artificial intelligence, scientific computing, and more.

Python code is executed line by line, making it easy to write and debug. It uses indentation to define code blocks, enhancing code readability.

Getting Started with Python

To start coding in Python, you'll need to install Python on your computer. You can download it from the official Python website. Once installed, you can write Python code using a text editor or an Integrated Development Environment (IDE) like PyCharm, VSCode, or Jupyter Notebook.

Python Syntax

Python syntax is designed to be readable and straightforward. Here is a simple Python program: 

# This is a comment
print("Hello, World!")  # This line 
 prints a message

Python Variables

Variables in Python are used to store data. You don't need to declare the type of a variable explicitly, as Python is dynamically typed. 

# Variable assignment
x = 5
y = "Hello, Python"

# Print variables
print(x)
print(y)

Python Data Types

Python has several built-in data types, including numbers, strings, lists, tuples, sets, and dictionaries. 

# Different data types
num = 10             # Integer
pi = 3.14            # Float
name = "Python"      # String
is_cool = True       # Boolean

# Print data types
print(type(num))
print(type(pi))
print(type(name))
print(type(is_cool))

Python Functions

Functions in Python are defined using the def keyword. They help in organizing code into reusable blocks. 

# Function definition
def greet(name):
    return f"Hello, {name}!"

# Function call
message = greet("World")
print(message)

Python Control Structures

Python provides various control structures like conditionals and loops to control the flow of your program. 

# If-else statement
age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

# For loop
for i in range(5):
    print(i)

# While loop
count = 0
while count < 5:
    print(count)
    count += 1
