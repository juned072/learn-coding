Introduction to C Programming

C is a general-purpose programming language that is extremely popular, simple, and flexible. It allows you to build low-level programming operations and has a wide range of applications.

Getting Started

Before you start coding in C, ensure you have a C compiler installed on your system. You can use GCC, which is a widely-used compiler, or any other compiler that suits your need.

Here is a simple C program to get you started:


#include <stdio.h>
int main() {
printf("Hello, World!");
return 0;
}

C Syntax

The syntax of C includes several elements like keywords, operators, and data types. Below are some basics:

Keywords

Keywords are reserved words that have special meanings in C. Examples include int, return, if, and else.

Operators

Operators are symbols that perform operations on variables and values. Common operators include + (addition), - (subtraction), and = (assignment).

Data Types

Data types specify the type of data that a variable can hold. Common data types in C include int, float, char, and double.

Control Structures

Control structures in C allow you to control the flow of the program. They include conditional statements like if, else, switch, and loops like for, while, and do-while.

Conditional Statements


#include <stdio.h>

int main() {
    int num = 10;
if (num > 0) {
  printf("Number is positive");
  } else {
  printf("Number is not positive");
    }

    return 0;
}

Loops


#include <stdio.h>

int main() {
    int i;
  for (i = 0; i < 5; i++) {
      printf("%d ", i);
    }
    return 0;
}

Functions

Functions in C allow you to break down your code into smaller, reusable components. You can define functions to perform specific tasks and call them as needed. 


#include <stdio.h>

void greet() {
    printf("Hello, World!");
}

int main() {
    greet();
    return 0;
}

Pointers

Pointers are variables that store the memory address of another variable. They are powerful but require careful handling. 


#include <stdio.h>

int main() {
  int var = 20;
  int *ptr;
  ptr = var;

printf("Value of var: %d\n", var);
printf("Address of var: %p\n", ptr);
printf("Value at the address 
 stored in ptr: %d\n", *ptr);

    return 0;
}

Arrays
