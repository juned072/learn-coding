Introduction to C++ Programming

C++ is a general-purpose programming language created as an extension of the C programming language, or "C with Classes". It has imperative, object-oriented and generic programming features.

Getting Started

Before you start coding in C++, ensure you have a C++ compiler installed on your system. You can use GCC, which is a widely-used compiler, or any other compiler that suits your need.

Here is a simple C++ program to get you started:


#include <iostream>

int main() {
    std::cout << "Hello, 
 World!" << std::endl;
    return 0;
}

C++ Syntax

The syntax of C++ includes several elements like keywords, operators, and data types. Below are some basics:

Keywords

Keywords are reserved words that have special meanings in C++. Examples include int, return, if, else, while, for, etc.

Operators

Operators are symbols that perform operations on variables and values. Common operators include + (addition), - (subtraction), = (assignment).

Data Types

Data types specify the type of data that a variable can hold. Common data types in C++ include int, float, char, double, and bool.

Control Structures

Control structures in C++ allow you to control the flow of the program. They include conditional statements like if, else, switch, and loops like for, while, and do-while.

Conditional Statements


#include <iostream>

int main() {
    int num = 10;

    if (num > 0) {
        std::cout << "Number 
 is positive" << std::endl;
    } else {
        std::cout << "Number 
 is not positive" << std::endl;
    }

    return 0;
}

Loops


#include <iostream>

int main() {
    for (int i = 0; i < 5; i++) {
        std::cout << i << " ";
    }
    std::cout << std::endl;

    return 0;
}

Functions

Functions in C++ allow you to break down your code into smaller, reusable components. You can define functions to perform specific tasks and call them as needed. 


#include <iostream>

void greet() {
    std::cout << "Hello, World!" 
 << std::endl;
}

int main() {
    greet();
    return 0;
}

Arrays

Arrays in C++ allow you to store multiple values of the same type in a single variable. You can access elements in an array using their index. 


#include <iostream>

int main() {
  int numbers[5] = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        std::cout << 
 numbers[i] << " ";
    }
    std::cout << std::endl;

    return 0;
}

Pointers

Pointers are variables that store the memory address of another variable. They are powerful but require careful handling. 


#include <iostream>

int main() {
    int var = 20;
    int *ptr = var;

    std::cout << "Value of var: 
 " << var << std::endl;
    std::cout << "Address of var: 
 " << ptr << std::endl;
    std::cout << "Value at the 
 address stored in ptr: 
 " << *ptr << std::endl;

    return 0;
}

Classes and Objects

Classes are user-defined data types that act as blueprints for objects. Objects are instances of classes. 


#include <iostream>

class Car {
public:
    std::string brand;
    std::string model;
    int year;

    void display() {
        std::cout << "Brand: " 
 << brand << ", 
 Model: " << model 
 << ", Year: " << year << std::endl;
    }
};

int main() {
    Car car1;
    car1.brand = "Toyota";
    car1.model = "Corolla";
    car1.year = 2020;

    car1.display();

    return 0;
}

File I/O

C++ provides functions to handle file operations like reading, writing, and closing files. 


#include <fstream>
#include <iostream>

int main() {
    std::ofstream file("test.txt");

    if (file.is_open()) {
        file << "Hello, World!";
        file.close();
    }

    std::ifstream fileRead 
("test.txt");
    std::string line;
    if (fileRead.is_open()) {
        while (getline 
(fileRead, line)) {
            std::cout << 
 line << std::endl;
        }
        fileRead.close();
    }

    return 0;
}
