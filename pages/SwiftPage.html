Introduction to Swift Programming

Swift is a powerful and intuitive programming language for iOS, macOS, watchOS, and tvOS app development. Writing Swift code is interactive and fun, the syntax is concise yet expressive, and Swift includes modern features developers love.

Getting Started

Before you start coding in Swift, ensure you have Xcode installed on your macOS. Xcode is Apple's integrated development environment (IDE) for macOS.

Here is a simple Swift program to get you started:


import Foundation

print("Hello, World!")

Swift Syntax

The syntax of Swift includes several elements like keywords, operators, and data types. Below are some basics:

Keywords

Keywords are reserved words that have special meanings in Swift. Examples include var, let, func, if, else, while, for, etc.

Operators

Operators are symbols that perform operations on variables and values. Common operators include + (addition), - (subtraction), = (assignment).

Data Types

Data types specify the type of data that a variable can hold. Common data types in Swift include Int, Float, Double, String, and Bool.

Control Flow

Control flow statements in Swift allow you to control the flow of the program. They include conditional statements like if, else, switch, and loops like for, while, and repeat-while.

Conditional Statements


var num = 10

if num > 0 {
    print("Number is positive")
} else {
    print("Number is not positive")
}

Loops


for i in 0..<5 {
    print(i)
}

Functions

Functions in Swift allow you to encapsulate a piece of code that you can call and reuse. Functions can have parameters and return values. 


func greet() {
    print("Hello, World!")
}

greet()

Arrays

Arrays in Swift allow you to store multiple values of the same type in a single variable. You can access elements in an array using their index. 


let numbers = [1, 2, 3, 4, 5]

for number in numbers {
    print(number)
}

Optionals

Optionals are used in Swift to handle the absence of a value. An optional either contains a value or contains nil to indicate that a value is missing. 


var name: String? = "John Doe"

if let unwrappedName = name {
    print("The name 
 is \(unwrappedName)")
} else {
    print("No name provided")
}

Classes and Objects

Classes are user-defined data types that act as blueprints for objects. Objects are instances of classes. 


class Car {
    var brand: String
    var model: String
    var year: Int

    init(brand: String, model: 
 String, year: Int) {
        self.brand = brand
        self.model = model
        self.year = year
    }

    func display() {
        print("Brand: \(brand), 
 Model: \(model), 
 Year: \(year)")
    }
}

let car1 = Car(brand: 
 "Toyota", model: 
 "Corolla", year: 2020)
car1.display()

Error Handling

Swift provides error handling to deal with unexpected conditions. You use the throw, try, and catch keywords to handle errors in Swift. 


enum DivisionError: Error {
    case divisionByZero
}

func divide(_ numerator: 
 Double, by denominator: 
 Double) throws -> Double {
 if denominator == 0 {
 throw DivisionError.divisionByZero
    }
    return numerator / denominator
}

do {
    let result = try 
 divide(10, by: 0)
    print(result)
} catch DivisionError. 
 divisionByZero {
    print("Error: Division by zero")
}
