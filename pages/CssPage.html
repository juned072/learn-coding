Introduction to CSS

CSS (Cascading Style Sheets) is used to control the style and layout of web pages. CSS allows you to apply styles to HTML elements and control their appearance.

CSS can be added to HTML documents in three ways: Inline, Internal, and External.

CSS Syntax

CSS is written in the form of rules, which consist of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

selector {
  property: value;
}

CSS Selectors

CSS selectors are used to select the HTML elements you want to style. Here are some common selectors:

  • element - Selects all elements of the specified type.
  • .class - Selects all elements with the specified class.
  • #id - Selects the element with the specified id.

CSS Colors

Colors in CSS can be defined using a color name, an RGB value, a HEX value, or other color formats. Here are some examples:

  • color: red; - Using a color name.
  • color: rgb(255, 0, 0); - Using an RGB value.
  • color: #ff0000; - Using a HEX value.

CSS Backgrounds

CSS backgrounds are used to control the background of elements. Here are some properties you can use:

  • background-color - Sets the background color.
  • background-image - Sets the background image.
  • background-repeat - Sets if/how the background image will repeat.
  • background-size - Sets the size of the background image.

CSS Borders

CSS borders are used to create borders around elements. Here are some properties you can use:

  • border-width - Sets the width of the border.
  • border-style - Sets the style of the border (e.g., solid, dashed).
  • border-color - Sets the color of the border.

CSS Margins

CSS margins are used to create space around elements, outside of any defined borders. Here are some properties you can use:

  • margin-top - Sets the top margin.
  • margin-right - Sets the right margin.
  • margin-bottom - Sets the bottom margin.
  • margin-left - Sets the left margin.

CSS Padding

CSS padding is used to create space around an element's content, inside of any defined borders. Here are some properties you can use:

  • padding-top - Sets the top padding.
  • padding-right - Sets the right padding.
  • padding-bottom - Sets the bottom padding.
  • padding-left - Sets the left padding.

CSS Text

CSS provides a number of properties to control the appearance of text. Here are some common properties:

  • color - Sets the color of the text.
  • text-align - Sets the horizontal alignment of the text.
  • text-decoration - Sets the decoration of the text (e.g., underline).
  • font-size - Sets the size of the text.
  • font-family - Sets the font of the text.

CSS Display

The display property specifies how an element is displayed. Here are some common values:

  • none - The element will not be displayed.
  • block - The element is displayed as a block element.
  • inline - The element is displayed as an inline element.
  • flex - The element is displayed as a flex container.

CSS Positioning

CSS positioning is used to position elements in a web page. Here are some properties you can use:

  • position - Sets the type of positioning (e.g., static, relative, absolute, fixed).
  • top - Sets the top position.
  • right - Sets the right position.
  • bottom - Sets the bottom position.
  • left - Sets the left position.

CSS Flexbox

CSS Flexbox is a layout model that allows you to design complex layouts more easily. Here are some properties you can use:

  • display: flex; - Sets the element as a flex container.
  • flex-direction - Defines the direction of the flex items (e.g., row, column).
  • justify-content - Aligns the flex items along the main axis.
  • align-items - Aligns the flex items along the cross axis.
